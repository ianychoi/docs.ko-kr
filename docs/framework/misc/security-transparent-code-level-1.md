- <span data-ttu-id="d86a6-120">보안 투명 코드는 <xref:System.Security.Permissions.SecurityAction.Assert> 작업을 수행할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d86a6-120">Security-transparent code cannot perform <xref:System.Security.Permissions.SecurityAction.Assert> actions.</span></span>
