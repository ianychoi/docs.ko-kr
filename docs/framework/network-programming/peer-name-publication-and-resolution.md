 <span data-ttu-id="abf7f-113">PNRP 네트워크 또는 클라우드에서 다른 피어를 찾는 것은 다음 두 단계로 구성된 프로세스입니다.</span><span class="sxs-lookup"><span data-stu-id="abf7f-113">Locating other peers in a PNRP network or cloud is a process comprised of two phases:</span></span>
