## <a name="creating-the-custom-class"></a><span data-ttu-id="ebc66-105">사용자 지정 클래스 만들기</span><span class="sxs-lookup"><span data-stu-id="ebc66-105">Creating the custom class</span></span>  

 <span data-ttu-id="ebc66-106">비제네릭 ForEach 클래스는 자식 작업을 예약해야 하고 <xref:System.Activities.NativeActivity>에서 파생되는 작업에는 이 기능이 없기 때문에 <xref:System.Workflow.Activities.CodeActivity>에서 파생되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-106">Since the Non-Generic ForEach class will need to schedule child activities, it will need to derive from <xref:System.Activities.NativeActivity>, since activities that derive from <xref:System.Workflow.Activities.CodeActivity> do not have this functionality.</span></span>  
  
```csharp  
public sealed class ForEach : NativeActivity  
{
}
```  
  
 <span data-ttu-id="ebc66-107">사용자 지정 클래스에서 작업에 사용되는 데이터를 저장하고 작업의 자식 작업을 실행하는 기능을 제공하려면 여러 멤버가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-107">The custom class requires several members to store data being used by the activity, and to provide functionality to execute the activity’s child activities.</span></span> <span data-ttu-id="ebc66-108">여기에는 다음과 같은 멤버가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-108">These members include:</span></span>  
  
- <span data-ttu-id="ebc66-109">`valueEnumerator`: 항목 컬렉션을 반복하는 데 사용되는 <xref:System.Activities.Variable%601> 형식의 public이 아닌 <xref:System.Collections.IEnumerator>입니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-109">`valueEnumerator`: The non-public <xref:System.Activities.Variable%601> of type <xref:System.Collections.IEnumerator> used to iterate over the collection of items.</span></span> <span data-ttu-id="ebc66-110">이 멤버는 작업에서 내부적으로 사용되기 때문에 이 개체의 원본이 작업 외부에 있는 경우에 사용되는 인수나 공용 속성 대신 <xref:System.Activities.Variable%601> 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-110">This member is of type <xref:System.Activities.Variable%601> because it is used internally in the activity, rather than an argument or public property, which would be used if this object were to have an origin outside the activity.</span></span>  
  
- <span data-ttu-id="ebc66-111">`OnChildComplete`: 각 자식의 실행이 완료될 때 실행되는 공용 <xref:System.Activities.CompletionCallback> 속성입니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-111">`OnChildComplete`: The public <xref:System.Activities.CompletionCallback> property that executes when each child completes execution.</span></span> <span data-ttu-id="ebc66-112">이 멤버는 작업의 각 인스턴스에 대해 값이 변경되지 않으므로 CLR 속성으로 정의됩니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-112">This member is defined as a CLR property, since its value will not change for different instances of the activity.</span></span>  
  
- <span data-ttu-id="ebc66-113">`Values`: 자식 작업의 반복에 사용되는 입력 컬렉션입니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-113">`Values`: The collection of inputs used for the iterations of the child activity.</span></span> <span data-ttu-id="ebc66-114">이 멤버는 데이터 원본이 작업 외부에 있지만 작업이 실행되는 동안 컬렉션 콘텐츠가 변경되지 않아야 하므로 <xref:System.Activities.InArgument%601> 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-114">This member is of type <xref:System.Activities.InArgument%601>, since the origin of the data is outside the activity, but the contents of the collection is not expected to change during the execution of the activity.</span></span> <span data-ttu-id="ebc66-115">작업이 실행되는 동안 이 컬렉션의 콘텐츠를 변경하는 기능(예: 작업 추가 또는 제거)이 작업에 필요한 경우 이 멤버는 <xref:System.Activities.ActivityAction>으로 정의된 다음 액세스할 때마다 평가되므로 작업에서 변경 내용을 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-115">If the activity needed the functionality to change the contents of this collection while the activity was executing (to add or remove activities, for instance), this member would have been defined as an <xref:System.Activities.ActivityAction>, which then would have been evaluated every time it was accessed, so that changes would be available to the activity.</span></span>  
  
- <span data-ttu-id="ebc66-116">`Body`: 이 멤버는 `Values` 컬렉션의 각 항목에 대해 실행할 작업을 정의합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-116">`Body`: This member defines the activity to be executed for each item in the `Values` collection.</span></span> <span data-ttu-id="ebc66-117">이 멤버는 액세스할 때마다 평가되도록 <xref:System.Activities.ActivityAction>으로 정의됩니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-117">This member is defined as an <xref:System.Activities.ActivityAction> so that it is evaluated every time it is accessed.</span></span>  
  
- <span data-ttu-id="ebc66-118">`Execute`: 이 메서드는 public이 아닌 `InternalExecute`, `OnForEachComplete` 및 `GetStateAndExecute` 멤버를 사용하여 Body 멤버에서 정의된 자식 작업의 실행을 예약하고 완료 처리기를 할당합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-118">`Execute`: This method uses the `InternalExecute`, `OnForEachComplete`, and `GetStateAndExecute` non-public members to schedule the execution and assign the completion handler of the child activity defined in the Body member.</span></span>  
  
- <span data-ttu-id="ebc66-119">`CacheMetadata`: 이 멤버는 작업을 실행하는 데 필요한 정보를 런타임에 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-119">`CacheMetadata`: This member provides the runtime with the information it needs to execute the activity.</span></span> <span data-ttu-id="ebc66-120">기본적으로 작업의 `CacheMetadata` 메서드가 작업의 모든 공용 멤버를 런타임에 알리지만, 이 작업은 일부 기능에 전용 멤버를 사용하기 때문에 런타임에서 인식할 수 있도록 전용 멤버의 존재를 런타임에 알려야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-120">By default, an activity’s `CacheMetadata` method will inform the runtime of all public members of the activity, but since this activity uses private members for some functionality, it needs to inform the runtime of their existence so that the runtime can be aware of them.</span></span> <span data-ttu-id="ebc66-121">이 경우 전용 `CacheMetadata` 멤버에 액세스할 수 있도록 `valueEnumerator` 함수가 재정의됩니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-121">In this case, the `CacheMetadata` function is overridden so that the private `valueEnumerator` member can be accessed.</span></span> <span data-ttu-id="ebc66-122">또한 이 멤버는 실행되는 동안 작업에 전달될 수 있도록 작업 값에 대한 인수를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="ebc66-122">This member also creates an argument for the values for the activity so that they can be passed in to the activity during execution.</span></span>
