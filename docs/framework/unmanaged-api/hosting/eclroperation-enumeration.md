 <span data-ttu-id="7257f-122">반면, *중요 하지 않은 코드 영역은* CLR에서 중단 또는 실패가 오류가 발생 한 작업에만 영향을 줄 수 있도록 보장 하는 영역입니다.</span><span class="sxs-lookup"><span data-stu-id="7257f-122">A *non-critical region of code*, on the other hand, is a region where the CLR can guarantee that an abort or a failure will affect only the task upon which the error occurs.</span></span>
