## <a name="extending-addressing-in-wcf-services"></a><span data-ttu-id="9f0e9-160">WCF 서비스에서 주소 확장</span><span class="sxs-lookup"><span data-stu-id="9f0e9-160">Extending Addressing in WCF Services</span></span>
