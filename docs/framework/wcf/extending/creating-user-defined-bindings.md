### <a name="example"></a><span data-ttu-id="ec231-173">예제</span><span class="sxs-lookup"><span data-stu-id="ec231-173">Example</span></span>  

 <span data-ttu-id="ec231-174">이 예제에서는 `SampleProfileUdpBinding`에서 파생되는 <xref:System.ServiceModel.Channels.Binding>의 프로필 바인딩을 구현합니다.</span><span class="sxs-lookup"><span data-stu-id="ec231-174">This example implements profile binding in `SampleProfileUdpBinding`, which derives from <xref:System.ServiceModel.Channels.Binding>.</span></span> <span data-ttu-id="ec231-175">에는 `SampleProfileUdpBinding` 최대 4 개의 바인딩 요소가 포함 됩니다. 하나는 사용자가 만든 `UdpTransportBindingElement` 이 고 3 개의 시스템 제공: `TextMessageEncodingBindingElement` , `CompositeDuplexBindingElement` 및 `ReliableSessionBindingElement` 입니다.</span><span class="sxs-lookup"><span data-stu-id="ec231-175">The `SampleProfileUdpBinding` contains up to four binding elements within it: one user-created `UdpTransportBindingElement`; and three system-provided: `TextMessageEncodingBindingElement`, `CompositeDuplexBindingElement`, and `ReliableSessionBindingElement`.</span></span>  
  
```csharp

{
    BindingElementCollection bindingElements = new BindingElementCollection();  
    if (ReliableSessionEnabled)

        bindingElements.Add(session);

```
