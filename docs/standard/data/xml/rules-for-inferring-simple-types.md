## <a name="type-promotion"></a><span data-ttu-id="059dd-161">형식 승격</span><span class="sxs-lookup"><span data-stu-id="059dd-161">Type Promotion</span></span>  
 <span data-ttu-id="059dd-162"><xref:System.Xml.Schema.XmlSchemaInference> 클래스는 특성 및 요소 값을 한 번에 하나씩 검사합니다.</span><span class="sxs-lookup"><span data-stu-id="059dd-162">The <xref:System.Xml.Schema.XmlSchemaInference> class examines attribute and element values one at a time.</span></span> <span data-ttu-id="059dd-163">값이 나타나면 가장 제한되고 부호 없는 형식이 유추됩니다.</span><span class="sxs-lookup"><span data-stu-id="059dd-163">As values are encountered, the most restrictive, unsigned type is inferred.</span></span> <span data-ttu-id="059dd-164">특성 또는 요소에 대해 형식이 유추되었고 현재 유추된 값과 일치하지 않는 새로운 값을 발견하는 경우, 유추된 형식은 현재 유추된 형식 및 새 값에 모두 적용되는 새로운 형식으로 승격됩니다.</span><span class="sxs-lookup"><span data-stu-id="059dd-164">If a type has been inferred for an attribute or element, and a new value is encountered that does not match the currently inferred type, the inferred type is promoted to a new type that applies to both the currently inferred type and the new value.</span></span> <span data-ttu-id="059dd-165"><xref:System.Xml.Schema.XmlSchemaInference> 클래스는 유추된 형식을 승격할 때 이전 값을 고려합니다.</span><span class="sxs-lookup"><span data-stu-id="059dd-165">The <xref:System.Xml.Schema.XmlSchemaInference> class does consider previous values when promoting the inferred type.</span></span>
