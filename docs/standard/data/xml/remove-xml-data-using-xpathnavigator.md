### <a name="removing-a-node"></a><span data-ttu-id="655bd-110">노드 제거</span><span class="sxs-lookup"><span data-stu-id="655bd-110">Removing a Node</span></span>  

 <span data-ttu-id="655bd-111"><xref:System.Xml.XPath.XPathNavigator> 클래스는 XML 문서에서 <xref:System.Xml.XPath.XPathNavigator.DeleteSelf%2A> 개체가 현재 위치하고 있는 현재 노드를 삭제하는 <xref:System.Xml.XPath.XPathNavigator> 메서드를 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-111">The <xref:System.Xml.XPath.XPathNavigator> class provides the <xref:System.Xml.XPath.XPathNavigator.DeleteSelf%2A> method to delete the current node an <xref:System.Xml.XPath.XPathNavigator> object is currently positioned on from an XML document.</span></span>

 <span data-ttu-id="655bd-117">다음 예제에서는 `price` 메서드를 사용하여 `book` 파일의 첫 번째 `contosoBooks.xml` 요소에서 <xref:System.Xml.XPath.XPathNavigator.DeleteSelf%2A> 요소를 삭제합니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-117">In the following example, the `price` element of the first `book` element of the `contosoBooks.xml` file is deleted using the <xref:System.Xml.XPath.XPathNavigator.DeleteSelf%2A> method.</span></span> <span data-ttu-id="655bd-118"><xref:System.Xml.XPath.XPathNavigator> 요소를 삭제한 후 `price` 개체는 부모 `book` 요소에 배치됩니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-118">The position of the <xref:System.Xml.XPath.XPathNavigator> object after the `price` element is deleted is on the parent `book` element.</span></span>

### <a name="removing-typed-values"></a><span data-ttu-id="655bd-134">형식화된 값 제거</span><span class="sxs-lookup"><span data-stu-id="655bd-134">Removing Typed Values</span></span>  

 <span data-ttu-id="655bd-135">노드의 형식이 W3C XML 스키마 단순 형식이면 <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A> 메서드에서 삽입한 새 값은 설정되기 전에 단순 형식의 패싯에 대해 검사됩니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-135">When the type of a node is a W3C XML Schema simple type, the new value inserted by the <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A> method is checked against the facets of the simple type before the value is set.</span></span> <span data-ttu-id="655bd-136">노드의 형식에 따라 새 값이 유효하지 않은 경우, 예를 들면 `-1` 형식의 요소에 `xs:positiveInteger`의 값을 설정하는 경우에는 예외가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-136">If the new value is not valid according to the type of the node (for example, setting a value of `-1` on an element whose type is `xs:positiveInteger`), it results in an exception.</span></span> <span data-ttu-id="655bd-137">또한 <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A>을 매개 변수로 `null` 메서드에 전달할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-137">The <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A> method also cannot be passed `null` as a parameter.</span></span> <span data-ttu-id="655bd-138">결과적으로 형식화된 노드 값을 제거하려면 노드의 스키마 형식을 따라야 합니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-138">As a result removing the value of a typed node must comply with the schema type of the node.</span></span>  
  
 <span data-ttu-id="655bd-139">다음 예제에서는 값을 `price`으로 설정하여 `book` 메서드로 `contosoBooks.xml` 파일에서 첫 번째 <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A> 요소의 `0` 요소 값을 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-139">The following example removes the value of the `price` element of the first `book` element in the `contosoBooks.xml` file using the <xref:System.Xml.XPath.XPathNavigator.SetTypedValue%2A> method by setting the value to `0`.</span></span> <span data-ttu-id="655bd-140">노드 값은 제거되지 않지만 `xs:decimal`의 데이터 형식에 따라 책 가격이 제거됩니다.</span><span class="sxs-lookup"><span data-stu-id="655bd-140">The value of the node is not removed, but the price of the book has been removed according to its data type of `xs:decimal`.</span></span>

```vb  
Dim document As XmlDocument = New XmlDocument()  
document.Load(reader)  
Dim navigator As XPathNavigator = document.CreateNavigator()  
  
navigator.MoveToChild("bookstore", "http://www.contoso.com/books")  
navigator.MoveToChild("book", "http://www.contoso.com/books")

navigator.MoveToRoot()  
Console.WriteLine(navigator.OuterXml)  
```
