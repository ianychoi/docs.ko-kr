<span data-ttu-id="c2469-104">표준 숫자 서식 문자열은 일반 숫자 형식의 서식을 지정하는 데 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="c2469-104">Standard numeric format strings are used to format common numeric types.</span></span> <span data-ttu-id="c2469-105">표준 숫자 서식 문자열은 `Axx` 형식을 취합니다. 여기서</span><span class="sxs-lookup"><span data-stu-id="c2469-105">A standard numeric format string takes the form `Axx`, where:</span></span>

  - <span data-ttu-id="c2469-116">**.NET Framework 및.NET Core에서 .NET Core 2.0까지** 런타임은 최소 유효 숫자(즉, <xref:System.MidpointRounding.AwayFromZero?displayProperty=nameWithType> 사용)를 사용하여 결과를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="c2469-116">**On .NET Framework and .NET Core up to .NET Core 2.0**, the runtime selects the result with the greater least significant digit (that is, using <xref:System.MidpointRounding.AwayFromZero?displayProperty=nameWithType>).</span></span>

  > [!NOTE]
  > <span data-ttu-id="c2469-118">전체 자릿수 지정자는 결과 문자열의 자릿수를 결정합니다.</span><span class="sxs-lookup"><span data-stu-id="c2469-118">The precision specifier determines the number of digits in the result string.</span></span> <span data-ttu-id="c2469-119">선행 또는 후행 공백을 포함한 결과 문자열을 채우려면 서식 항목에서 [복합 서식 지정](composite-formatting.md) 기능을 사용하고 *맞춤 구성 요소* 를 정의합니다.</span><span class="sxs-lookup"><span data-stu-id="c2469-119">To pad a result string with leading or trailing spaces, use the [composite formatting](composite-formatting.md) feature and define an *alignment component* in the format item.</span></span>
