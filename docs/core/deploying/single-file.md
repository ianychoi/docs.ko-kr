## <a name="publish-a-single-file-app---sample-project-file"></a><span data-ttu-id="f7859-158">단일 파일 앱 게시 - 샘플 프로젝트 파일</span><span class="sxs-lookup"><span data-stu-id="f7859-158">Publish a single file app - sample project file</span></span>

```xml

    <PublishReadyToRun>true</PublishReadyToRun>
  </PropertyGroup>
```
