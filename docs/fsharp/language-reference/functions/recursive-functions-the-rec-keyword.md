* <span data-ttu-id="e6770-123">`loop`자연 스러운 F # 패턴 인 이라는 재귀적 내부 함수입니다.</span><span class="sxs-lookup"><span data-stu-id="e6770-123">A recursive inner function named `loop`, which is an idiomatic F# pattern.</span></span>
