<span data-ttu-id="41cb5-104">`Structure` 의 LINQ 쿼리에는 `Me` 에 대한 참조 또는 구조체의 인스턴스 멤버에 대한 참조가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="41cb5-104">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="41cb5-105">`Me` 또는 인스턴스 멤버에 대한 참조는 `Structure`내의 쿼리 식에서 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="41cb5-105">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="41cb5-107">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="41cb5-107">To correct this error</span></span>  
  
1. <span data-ttu-id="41cb5-108">다음 예제와 같이 `Me` 에 대한 참조에서 반환된 값 또는 인스턴스 멤버의 복사본을 만들고 쿼리 식에서 복사본을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="41cb5-108">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
