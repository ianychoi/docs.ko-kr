<span data-ttu-id="76214-104">이 오류는 대리자와 메서드 간에 불가능한 변환이 필요한 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="76214-104">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="76214-105">오류의 원인은 매개 변수 간의 변환이거나, 메서드와 대리자가 함수일 때 반환 값의 변환일 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="76214-105">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>

 <span data-ttu-id="76214-108">다음 함수는 이 오류를 발생시키는 방법 면에서 각각 `FunDel` 과 다릅니다.</span><span class="sxs-lookup"><span data-stu-id="76214-108">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

```vb  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="76214-109">다음 각 대입문은 오류를 발생시킵니다.</span><span class="sxs-lookup"><span data-stu-id="76214-109">Each of the following assignment statements causes the error.</span></span>

```vb  
    'Dim d1 As FunDel = AddressOf ExampleMethod1

    ' are not compatible.
```  
