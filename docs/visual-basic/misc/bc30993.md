<span data-ttu-id="56c2a-104">개체 이니셜라이저 목록에서 초기화된 멤버는 필드 또는 속성 중 하나여야 합니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-104">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="56c2a-105">또한 이니셜라이저 목록의 속성은 매개 변수를 가질 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-105">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="56c2a-106">이 오류를 발생시키는 속성은 오버로드되며 해당하는 각 버전에 인수가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-106">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="56c2a-107">따라서 개체 이니셜라이저 목록의 속성을 초기화할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-107">Therefore, the property cannot be initialized in an object initializer list.</span></span>

- <span data-ttu-id="56c2a-110">이니셜라이저 목록에서 인수를 필요로 하는 속성을 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-110">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="56c2a-111">예</span><span class="sxs-lookup"><span data-stu-id="56c2a-111">Example</span></span>  

 <span data-ttu-id="56c2a-112">다음 클래스에는 `TotalItems` 에 대해 1개, `Item`(오버로드됨)에 대해 2개 등 총 3개의 속성 정의가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-112">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="56c2a-113">`TotalItems` 속성에는 인수가 필요하지 않으며 다음 선언과 같이 개체 초기화 목록에서 초기화될 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-113">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="56c2a-114">`Item` 속성은 오버로드되며 각 오버로드에는 인수가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-114">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="56c2a-115">따라서 `Item` 은 개체 이니셜라이저 목록에 나타날 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-115">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="56c2a-116">이 오류를 방지하려면 개체 이니셜라이저 밖에서 `Item` 속성을 초기화합니다.</span><span class="sxs-lookup"><span data-stu-id="56c2a-116">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
