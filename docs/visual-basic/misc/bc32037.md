<span data-ttu-id="1ea2e-104">다음 예제와 같이 중첩된 루프가 중첩에 있는 루프보다 많은 루프 변수를 지정하는 단일 `Next` 문으로 종료되었습니다.</span><span class="sxs-lookup"><span data-stu-id="1ea2e-104">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>

```vb  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
