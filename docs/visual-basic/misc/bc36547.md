# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="43559-103">무명 형식 멤버 또는 속성 '\<propertyname>'이 이미 선언되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="43559-103">Anonymous type member or property '\<propertyname>' is already declared</span></span>

<span data-ttu-id="43559-104">속성 이름은 무명 형식 선언에서 한 번만 설정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="43559-104">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="43559-105">예를 들어 다음 선언은 잘못된 것입니다.</span><span class="sxs-lookup"><span data-stu-id="43559-105">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```
