## <a name="example"></a><span data-ttu-id="4d3ae-112">예</span><span class="sxs-lookup"><span data-stu-id="4d3ae-112">Example</span></span>

<span data-ttu-id="4d3ae-113">다음 예제에서는 `AttributeUsage` 특성을 사용하여 새 버전의 `ExtensionAttribute` 가 적용할 수 있는 프로그램 요소를 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="4d3ae-113">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="4d3ae-114">이 예제에서는 `AttributeTargets` 열거형의 세 멤버인 `Assembly`, `Class`및 `Method`를 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="4d3ae-114">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="4d3ae-115">이들 요소 중 하나를 생략하면 이 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="4d3ae-115">The omission of any one of these elements will cause this error.</span></span>

```vb
Namespace System.Runtime.CompilerServices
    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="4d3ae-116">또는 `ExtensionAttribute` 에서 `All` 의 `AttributeTargets`멤버를 모든 프로그램 요소에 적용하는 것을 허용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4d3ae-116">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

```xml
<AttributeUsage(AttributeTargets.All)>
```

<span data-ttu-id="4d3ae-117">다음 코드와 같이 `AttributeUsage` 줄을 삭제하면 같은 결과를 얻을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4d3ae-117">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>

```vb

    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```
