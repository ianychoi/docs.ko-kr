# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="2bf26-103">다차원 배열은 식 트리로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2bf26-103">Multi-dimensional array cannot be converted to an expression tree</span></span>

<span data-ttu-id="2bf26-104">대부분의 식은 식 트리로 변환할 수 있지만 다차원 배열은 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2bf26-104">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="2bf26-105">예를 들어 다음 코드는 이 오류를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2bf26-105">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
