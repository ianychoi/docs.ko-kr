## <a name="to-correct-this-error"></a><span data-ttu-id="c80da-112">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="c80da-112">To correct this error</span></span>

- <span data-ttu-id="c80da-113">매개 변수 배열에서 확장하려는 데이터 형식을 지정하지 않는 경우 이 형식을 지정하는 새로운 첫 번째 매개 변수를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="c80da-113">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>

  ```vb

  Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)
  ' Concatenate the strings in addOns to str.
  End Sub
  ```

- <span data-ttu-id="c80da-114">매개 변수 배열에서 확장하려는 데이터 형식을 지정하는 경우 매개 변수 배열이 아니라 인수가 필요한 일반 배열로 변경하는 것이 좋습니다.</span><span class="sxs-lookup"><span data-stu-id="c80da-114">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="c80da-115">일반 배열은 확장할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="c80da-115">Regular arrays can be extended.</span></span>

  ```vb

  Public Function Sum(ByVal ints() As Integer) As Integer

      For Each i As Integer In ints
          total = total + i
      Next i
      Return total
  End Function
  ```
