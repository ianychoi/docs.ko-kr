## <a name="example"></a><span data-ttu-id="01a8c-110">예</span><span class="sxs-lookup"><span data-stu-id="01a8c-110">Example</span></span>  

 <span data-ttu-id="01a8c-111">다음 클래스에는 정수 인수가 필요한 기본 속성 `defaultProp`가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="01a8c-111">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="01a8c-112">기본 속성에 인수가 필요하기 때문에 다음 선언에서 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="01a8c-112">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```vb

```
