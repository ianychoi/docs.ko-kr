## <a name="example-3"></a><span data-ttu-id="8ce26-157">예제 3</span><span class="sxs-lookup"><span data-stu-id="8ce26-157">Example 3</span></span>

<span data-ttu-id="8ce26-158">다음 수정 된 `Person` 예제에서는 `other` 복사 생성자에 전달 된 개체가 먼저 메서드에서 null 인지 확인 합니다 `PassThroughNonNull` .</span><span class="sxs-lookup"><span data-stu-id="8ce26-158">In the following revised `Person` example, the `other` object that's passed to the copy constructor is first checked for null in the `PassThroughNonNull` method.</span></span>

```csharp
public class Person
{
    public string Name { get; private set; }
    public int Age { get; private set; }

    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    // Copy constructor
    public Person(Person other)
        : this(PassThroughNonNull(other).Name, other.Age)
    {
    }

    // Null check method
    private static Person PassThroughNonNull(Person person)
    {
        if (person == null)
            throw new ArgumentNullException(nameof(person));
        return person;
    }
}
```
