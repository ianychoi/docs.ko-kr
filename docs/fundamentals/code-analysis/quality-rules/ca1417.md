## <a name="how-to-fix-violations"></a><span data-ttu-id="d1cc5-116">위반 문제를 해결하는 방법</span><span class="sxs-lookup"><span data-stu-id="d1cc5-116">How to fix violations</span></span>

```csharp
 // Violation
[DllImport("MyLibrary")]
private static extern void Foo([Out] string s);

// Fixed: passed by reference
[DllImport("MyLibrary")]
private static extern void Foo(out string s);

// Fixed: marshaling data back to caller is not required
[DllImport("MyLibrary")]
private static extern void Foo(string s);
```
