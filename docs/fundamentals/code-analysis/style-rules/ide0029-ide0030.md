### <a name="example"></a><span data-ttu-id="632d9-134">예제</span><span class="sxs-lookup"><span data-stu-id="632d9-134">Example</span></span>

```csharp
// dotnet_style_coalesce_expression = true
var v = x ?? y;

// dotnet_style_coalesce_expression = false

var v = x == null ? y : x;
```

