#### <a name="example"></a><span data-ttu-id="a3130-127">예제</span><span class="sxs-lookup"><span data-stu-id="a3130-127">Example</span></span>

```csharp

Console.WriteLine($"{name} {age}");

Console.WriteLine($"{person.name} {person.age}");
```
