## <a name="example"></a><span data-ttu-id="3374d-109">예제</span><span class="sxs-lookup"><span data-stu-id="3374d-109">Example</span></span>  

 <span data-ttu-id="3374d-110">다음 샘플에서는 CS3018을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3374d-110">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
