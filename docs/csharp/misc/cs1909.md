## <a name="example"></a><span data-ttu-id="50b9e-106">예제</span><span class="sxs-lookup"><span data-stu-id="50b9e-106">Example</span></span>  

 <span data-ttu-id="50b9e-107">다음 샘플에서는 CS1909를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="50b9e-107">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
