## <a name="example"></a><span data-ttu-id="95391-109">예제</span><span class="sxs-lookup"><span data-stu-id="95391-109">Example</span></span>  

 <span data-ttu-id="95391-110">다음 코드에서는 CS1928 및 CS1929을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="95391-110">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
