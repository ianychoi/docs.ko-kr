## <a name="example"></a><span data-ttu-id="dfd03-109">예제</span><span class="sxs-lookup"><span data-stu-id="dfd03-109">Example</span></span>  

 <span data-ttu-id="dfd03-110">다음 샘플에서는 CS1666을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="dfd03-110">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
