<span data-ttu-id="97418-104">'type'은 이 어셈블리 외부에 노출되지 않으므로 CLS 규격 검사를 수행하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="97418-104">CLS compliance checking will not be performed on 'type' because it is not visible from outside this assembly.</span></span>  
  
 <span data-ttu-id="97418-105">이 경고는 <xref:System.CLSCompliantAttribute> 특성이 있는 형식이나 멤버가 다른 어셈블리에서 표시되지 않을 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="97418-105">This warning occurs when a type or a member that has the <xref:System.CLSCompliantAttribute> attribute is not visible from another assembly.</span></span> <span data-ttu-id="97418-106">이 오류를 해결하려면 다른 어셈블리에서 표시되지 않는 모든 클래스나 멤버에서 해당 특성을 제거하거나 해당 형식 또는 멤버가 표시되도록 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="97418-106">To resolve this error, remove the attribute on any classes or members that are not visible from the other assembly, or make the type or members visible.</span></span> <span data-ttu-id="97418-107">CLS 규격에 대 한 자세한 내용은 [언어 독립성 및 언어 독립적 구성 요소](../../standard/language-independence.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="97418-107">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="97418-108">예제</span><span class="sxs-lookup"><span data-stu-id="97418-108">Example</span></span>  

 <span data-ttu-id="97418-109">다음 샘플에서는 CS3019를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="97418-109">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line

{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
