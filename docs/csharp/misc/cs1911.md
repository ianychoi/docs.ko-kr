# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="848b2-103">컴파일러 경고(수준 1) CS1911</span><span class="sxs-lookup"><span data-stu-id="848b2-103">Compiler Warning (level 1) CS1911</span></span>

<span data-ttu-id="848b2-104">무명 메서드, 람다 식, 쿼리 식 또는 반복기에서 'base' 키워드를 통해 'name' 멤버에 액세스하면 확인할 수 없는 코드가 만들어집니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-104">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="848b2-105">포함하는 형식의 도우미 메서드로 액세스를 이동하는 것이 좋습니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-105">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="848b2-106">반복기 또는 무명 메서드의 메서드 본문 내에서 `base` 키워드를 사용하여 가상 함수를 호출하면 확인할 수 없는 코드가 만들어집니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-106">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="848b2-107">확인할 수 없는 코드는 부분 신뢰 환경에서 실행되지 못합니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-107">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="848b2-108">CS1911에 대한 하나의 해결책은 가상 함수 호출을 도우미 함수로 옮기는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-108">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="848b2-109">예제</span><span class="sxs-lookup"><span data-stu-id="848b2-109">Example</span></span>  

 <span data-ttu-id="848b2-110">다음 샘플에서는 CS1911을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="848b2-110">The following sample generates CS1911.</span></span>  
  
```csharp  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
