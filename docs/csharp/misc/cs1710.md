<span data-ttu-id="f62d4-104">'type'의 XML 주석에 'parameter'에 대한 중복된 형식 매개 변수 태그가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="f62d4-104">XML comment on 'type' has a duplicate typeparam tag for 'parameter'</span></span>

## <a name="example"></a><span data-ttu-id="f62d4-106">예제</span><span class="sxs-lookup"><span data-stu-id="f62d4-106">Example</span></span>  

 <span data-ttu-id="f62d4-107">다음 코드는 CS1710 경고가 표시되게 합니다.</span><span class="sxs-lookup"><span data-stu-id="f62d4-107">The following code will cause warning CS1710 to appear.</span></span>  
  
```csharp  
// CS1710.cs  
// compile with: /doc:cs1710.xml  
// To resolve this warning, delete one of the duplicate <typeparam>'s.  
using System;  
class Stack<ItemType>  
{  
}  
  
/// <typeparam name="MyType">can be an int</typeparam>  
/// <typeparam name="MyType">can be an int</typeparam>  
class MyStackWrapper<MyType>  
{  
    // Open constructed type Stack<MyType>.  
    Stack<MyType> stack;  
    public MyStackWrapper(Stack<MyType> s)  
    {  
        stack = s;  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        // Closed constructed type Stack<int>.  
        Stack<int> stackInt = new Stack<int>();  
        MyStackWrapper<int> MyStackWrapperInt =  
            new MyStackWrapper<int>(stackInt);  
    }  
}  
```
