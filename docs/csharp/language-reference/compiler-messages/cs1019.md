# <a name="compiler-error-cs1019"></a><span data-ttu-id="fcd17-103">컴파일러 오류 CS1019</span><span class="sxs-lookup"><span data-stu-id="fcd17-103">Compiler Error CS1019</span></span>

<span data-ttu-id="fcd17-104">오버로드할 수 있는 단항 연산자가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-104">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="fcd17-105">오버로드된 단항 연산자처럼 보이는 항목이 선언되었지만 연산자가 없거나 시그니처의 잘못된 위치에 있습니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-105">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="fcd17-106">*단항 연산자* 는 단일 피연산자에서 작동하는 연산자입니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-106">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="fcd17-107">예를 들어 `++`는 단항 연산자입니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-107">For example, `++` is a unary operator.</span></span> <span data-ttu-id="fcd17-108">`operator` 키워드를 사용하고 연산자가 작동하는 형식의 단일 매개 변수를 지정하여 일부 단항 연산자를 오버로드할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-108">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="fcd17-109">예를 들어 `Temperature++`를 작성할 수 있도록 사용자 정의 클래스 `Temperature`에 대해 `++` 연산자를 오버로드하려는 경우 다음과 같은 방식으로 정의할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-109">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="fcd17-110">이 오류가 표시되면 연산자 자체가 없거나 시그니처의 잘못된 위치에 있다는 점을 제외하고 오버로드된 단항 연산자처럼 보이는 항목을 선언한 것입니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-110">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="fcd17-111">앞의 예제에 있는 시그니처에서 `++`를 제거하면 CS1019가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-111">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="fcd17-112">다음 코드에서는 CS1019를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fcd17-112">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }  
}  
```
